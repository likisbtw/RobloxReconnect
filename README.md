RobloxReconnect v1.0

Скрипт предназначен для поддержания работы игрового клиента в режиме 24/7. Он отслеживает состояние процесса и автоматически перезапускает игру при вылетах, закрытии окна или возникновении сетевых ошибок.

Возможности
- Поддержка нескольких аккаунтов: запуск нескольких аккаунтов Roblox одновременно.
- Автоматический реконект: автоматически перезапускает аккаунты, если их закрыли или кикнули.
- Мониторинг логов: обнаружение киков, отключений и ошибок античит системы в реальном времени.
- Интегрированная консоль: логирование через графический интерфейс с отметками времени.
- Один клик - сборка: легко собрать в отдельный исполняемый файл (.exe).

Установка
1. Установите Python (рекомендуется версия 3.10 или выше).
2. Запустите команду (в папке с проектом нажмите в адресную строку, введите "cmd", нажмите Enter и в консоли введите): "pip install -r requirements.txt"
3. Запустите "build.bat", чтобы создать .exe файл в папке "dist".
4. Запустите "RobloxReconnector.exe" из папки "dist".
5. Добавьте свои аккаунты (Cookie, ID плейса, VIP-ссылка(необязательно)).

Использование
- Запустите команду "python main.py", чтобы запустить приложение.
- Включите Multi-Instance, если хотите использовать несколько аккаунтов.
- Нажмите Start Watchdog, чтобы включить автоматический реконект.
